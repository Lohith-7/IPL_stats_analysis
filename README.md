\# IPL Stats Analyzer



A Python data analysis project that visualizes IPL player performance using real cricket statistics.



\## Features



\* Search for any player

\* Visualize runs scored per season

\* Compare runs and strike rate trends

\* Dynamic graph generation



\## Technologies Used



\* Python

\* Pandas

\* Matplotlib

\* Seaborn



\## Installation



```bash

git clone https://github.com/yourusername/IPLstatsProject

cd IPLstatsProject

pip install -r requirements.txt

```



\## Run the Project



```bash

python main.py

```



\## Example Output



Displays a graph showing:



\* Runs per season (bar chart)

\* Strike rate trend (line chart)
